## 🚀 Project Overview

EduManage addresses the challenges of manual academic record-keeping by providing a centralized, secure, and visually appealing digital hub.

### Core Philosophy
- **Security First**: Forced PDO prepared statements and bcrypt hashing.
- **User Centric**: High-fidelity dashboards tailored to specific user needs.
- **Modern Aesthetics**: Built with Tailwind CSS and premium glassmorphism effects.
- **Scalability**: Modular PHP structure for easy extension.
